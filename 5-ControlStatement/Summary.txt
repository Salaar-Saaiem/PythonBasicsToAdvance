CONTROL STATEMENTS IN PYTHON

Control statements are used to control the flow of execution of a program.

1. CONDITIONAL STATEMENTS

These statements are used for decision making.

a) if Statement
Executes code only when condition is true.

Example:
age = 18

if age >= 18:
    print("Eligible")


b) if-else Statement
Executes one block if condition is true and another if false.

Example:
marks = 30

if marks >= 35:
    print("Pass")
else:
    print("Fail")


c) if-elif-else Statement
Used to check multiple conditions.

Example:
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Grade C")


2. LOOPING STATEMENTS

These statements are used to repeat code.

a) for Loop
Used when number of repetitions is known.

Example:
for i in range(5):
    print(i)


b) while Loop
Runs while condition remains true.

Example:
count = 1

while count <= 3:
    print(count)
    count += 1


3. JUMP STATEMENTS

These statements control loop execution.

a) break Statement
Stops the loop immediately.

Example:
for i in range(5):
    if i == 3:
        break
    print(i)


b) continue Statement
Skips current iteration and moves to next iteration.

Example:
for i in range(5):
    if i == 2:
        continue
    print(i)


c) pass Statement
Does nothing and is used as a placeholder.

Example:
for i in range(3):
    pass


SUMMARY

1. if, else, elif → Decision making
2. for, while → Repetition/looping
3. break, continue, pass → Loop control
